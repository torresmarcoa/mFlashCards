# Overview

This app was designed to help students study using digital flashcards. Users can create and view cards, and organize them into categories so that related cards stay grouped together.

To use the app, you create a category, add flashcards to it, and then open a category to study its cards. This keeps study sessions focused on one topic at a time. It is also possible to study all flashcards on one study session.

The purpose of this app is to help students memorize concepts and definitions with a well-known study method while removing the need to carry physical cards. The phone becomes a portable, always-available study tool.
